# 🧑 Profile Card Project

## 📌 Overview

This project is a responsive, accessible Profile Card built using semantic HTML, CSS, and vanilla JavaScript.

It includes dynamic time rendering and is fully testable using `data-testid` attributes.

---

## 🚀 Live Demo

👉 [View Live Project](https://your-live-url.com)

---

## 🛠 Technologies Used

* HTML5 (Semantic structure)
* CSS3 (Flexbox & Responsive Design)
* JavaScript (Dynamic time using Date.now())

---

## ♿ Accessibility Features

* Semantic HTML elements (`article`, `nav`, `section`)
* Accessible image alt text
* Keyboard navigable links
* `aria-live` for dynamic time updates

---

## 🧪 Testability

All UI elements include `data-testid` attributes for automated testing.
